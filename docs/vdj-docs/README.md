# 📚 Documentación Luna VDJ Skin

Esta carpeta contiene toda la documentación reorganizada para el desarrollo del skin Luna para VirtualDJ, estructurada de manera lógica y progresiva.

## 🗂️ Estructura de la Documentación

### 📖 **00-general** - Conceptos Fundamentales
Introducción al desarrollo de skins para VirtualDJ, conceptos básicos y configuración inicial.

### 🔧 **01-desarrollo-basico** - Fundamentos del Desarrollo
Elementos básicos para crear skins: definiciones, posicionamiento, propiedades y grupos.

### 🎛️ **02-elementos-interfaz** - Controles e Interfaz
Elementos interactivos como botones, sliders, menús y zonas especiales.

### 🎨 **03-componentes-visuales** - Elementos Visuales
Componentes gráficos y visualizaciones: formas de onda, scratch, video, navegador, etc.

### 💾 **04-scripting-avanzado** - Programación Avanzada
VDJScript, acciones combinadas, LEDs RGB y funcionalidades avanzadas.

### 📋 **05-referencias-tecnicas** - Referencias y SDK
Documentación técnica, listas de referencia, SDK y recursos para desarrolladores.

## 🚀 Guía de Inicio Rápido

1. **Nuevo en desarrollo de skins**: Comienza por `00-general/`
2. **Conceptos básicos**: Continúa con `01-desarrollo-basico/`
3. **Elementos específicos**: Consulta `02-elementos-interfaz/` y `03-componentes-visuales/`
4. **Funcionalidades avanzadas**: Explora `04-scripting-avanzado/`
5. **Referencias rápidas**: Utiliza `05-referencias-tecnicas/`

## 📝 Carpetas de Usuario

### **01-guias-usuario** - Guías para Usuarios Finales
Documentación dirigida a usuarios que utilizan el skin.

### **01-manuales-usuario** - Manuales de Usuario
Manuales completos sobre funcionalidades y configuraciones.
